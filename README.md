# Clustering de Dados Astronômicos

Projeto acadêmico utilizando algoritmos de aprendizado não supervisionado
(Hierárquico, DBSCAN e GMM) para análise de classes espectrais estelares.

## Tecnologias
- Python
- Pandas
- Scikit-learn
- Matplotlib / Seaborn

## Resultados
Comparação dos algoritmos por Silhouette Score e métricas externas.
